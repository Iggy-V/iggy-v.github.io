Automated Daily Email

A daily check-in with a Google Form and some information about the world, the weather and a picture of a dog!

Overview

I wanted to somehow keep track of my mood, habits, sleep, etc., so that I would have data to play around with and get to better know myself. That's what this achieves.

I get a daily reminder to fill out a Google Form which has questions I want to track, top 3 news of the day, weather for the week and a cute picture of a dog to spark a smile.

Goals

  • Get a daily reminder to fill out a form
  • Deploy a program on AWS
  • Get very comfortable with APIs

Lessons Learned:

  • How to use AWS Lambda
  • How to deploy dependancies to AWS Lambda
  • How to hide variables like API keys
  • How to format and send emails using Python

dependencies:

  • Requests
  • EmailMessage

API:

  • The Dog API
  • Tomorrow API
  • News API
