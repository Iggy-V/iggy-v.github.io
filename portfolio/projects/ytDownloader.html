YouTube Downloader

Download all opened YouTube links to prepare yourself for the scary life without internet.

NOTE: The code is for educational purposes and if used to download content from YouTube it must follow community guidelines set by YouTube that can be found here: https://www.youtube.com/howyoutubeworks/policies/community-guidelines/

Overview

YouTube Downloader makes it easy to prepare for long flights or traveling to remote areas while staying in touch with your favorite content creators.

The project was inspired by frequent flying and also served particularly useful while travelling during the Semester at Sea program.

Goals

  • Be able to access YouTube videos without internet
  • Have a nice user interface that is easy to use for the user

Dependancies:

  • PyTube 15.0.0
  • PyGetWindow 0.0.9
  • pyperclip 1.8.2
  • PyAutoGUI 0.9.54
  • Chrome 115.0.5790.171

Lessons Learnt:

  • How to use PyTube to download YouTube videos
  • How to use PyGetWindow to get the window of the browser
  • How to use pyperclip to copy and paste the URL
  • How to use PyAutoGUI to automate the OS's GUI to emulate tasks of the user.

Future work:

  • Implement a way to download the videos in the background
  • Possibly make a Chrome extension
  • Automatically fetch all the new videos from the subscribed channels.
