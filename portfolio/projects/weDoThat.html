We Do That

Create, join and see the events surrounding you in your community. Connection has never been this easy!

Overview

A initial try of trying to create a website that would allow people to connect with their local communities.

This project was created during HooHacks 2024 in University of Virginia!

Goals

  • Create a web application that allows for people to log-in, post, and join different events
  • Implement a database to keep track of what has been posted and registered for
  • Implement searchability features, like search tab and filters

Lessons Learnt:

  • How to create a dynamic website with a database
  • MySQL and PHP
  • How to dynamically generate content

Future Work
