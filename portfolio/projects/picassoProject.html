Picasso Project

Create Art with the x and y coordinates!

Overview

Final project for my CSCI 209: Software Development class.

You just have to put in a mathematical expression that uses x and/or y and boom art unfold before your eyes.

Goals

  • Understanding code base
  • Implementing basic functionality
  • Adding unique extensions

Lessons Learnt:

  • How to code in Java and best Java practices
  • How to use Eclipse IDE
  • The importance of Single-Responsibility principle and DRY principle
  • How to add new inventive features to a code base given.

Personal Contribution:

  • Defined Variable GUI Extension - showcases what expression is currently being used.
  • History Toggle Extension - shows previously entered expressions and allows with the arrow keys to toggle between the different expressions.
  • Implemented multiple unary expressions like times and division.
